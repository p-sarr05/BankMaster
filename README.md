# 💳 BankMaster

## 📌 Description

BankMaster est une application web de gestion bancaire développée en HTML, CSS et JavaScript.
Elle simule le fonctionnement d’un système bancaire en permettant la gestion des comptes utilisateurs et des opérations financières.

---

## 🚀 Fonctionnalités

* 👤 Gestion des clients
* 💰 Création de comptes bancaires
* 🔄 Dépôt et retrait d’argent
* 🔁 Transfert entre comptes
* 📊 Historique des transactions
* 💾 Sauvegarde des données avec LocalStorage

---

## 🛠️ Technologies utilisées

* HTML5
* CSS3
* JavaScript

---

## ⚙️ Installation et utilisation

1. Télécharger ou cloner le projet :

```bash
git clone https://github.com/p-sarr05/BankMaster.git
```

2. Ouvrir le fichier `index.html` dans un navigateur

---

## 📁 Structure du projet

* `index.html` : page principale
* `style.css` : design de l’application
* `app.js` : logique principale
* `clients.js` : gestion des clients
* `comptes.js` : gestion des comptes
* `operations.js` : gestion des transactions
* `storage.js` : gestion du LocalStorage

---

## ⚠️ Limites

* Pas de backend (données stockées localement)
* Pas de sécurité avancée (authentification simulée)

---

## 🎯 Objectif du projet

Ce projet a été réalisé dans le cadre d’un apprentissage du développement web afin de maîtriser la manipulation du DOM, la gestion des données et la logique applicative en JavaScript.
